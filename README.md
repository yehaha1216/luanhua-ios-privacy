# 乱花 - 隐私政策网站

这是乱花 App 的隐私政策网站，托管在 GitHub Pages 上。

## 文件说明

- `index.html` - 隐私政策页面
- `icon.png` - 应用图标
- `README.md` - 本文件

## 部署到 GitHub Pages

1. 确保你已经在 GitHub 上创建了仓库
2. 推送代码到 GitHub：

```bash
cd luanhua-ios-privacy
git add .
git commit -m "Initial commit: Privacy policy website"
git push origin main
```

3. 在 GitHub 仓库设置中启用 GitHub Pages：
   - 进入仓库的 Settings
   - 找到 Pages 选项
   - 在 Source 下选择 `main` 分支
   - 点击 Save

4. 几分钟后，你的网站将在以下地址可用：
   `https://[你的用户名].github.io/[仓库名]/`

## 本地预览

直接在浏览器中打开 `index.html` 文件即可预览。

## 更新隐私政策

编辑 `index.html` 文件，然后推送更改到 GitHub：

```bash
git add index.html
git commit -m "Update privacy policy"
git push origin main
```
